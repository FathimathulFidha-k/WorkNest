## Hosting Commands
To deploy your application to Firebase Hosting, use the following commands:

1. Install Firebase CLI:
   ```bash
   npm install -g firebase-tools
   ```

2. Log in to your Firebase account:
   ```bash
   firebase login
   ```

3. Initialize Firebase in your project:
   ```bash
   firebase init
   ```

4. Deploy the app:
   ```bash
   firebase deploy
   ```
